# Indoor Positioning System Using Bluetooth RSSI and Trilateration

## Overview
This is a fork of the original code for an indoor positioning system using Bluetooth RSSI and trilateration. The code is designed to run on Raspberry pi.

## Usage
`pip install adafruit-circuitpython-minimqtt adafruit-circuitpython-ntp adafruit-circuitpython-ble`

### for windows 
```cmd
# Install pip (Usually included with Python installation on Windows)
# If not installed, download get-pip.py from official sources

# Create a virtual environment
python -m venv venv

# Activate the virtual environment
.\venv\Scripts\activate

# Install the required libraries
pip install adafruit-circuitpython-minimqtt adafruit-circuitpython-ntp adafruit-circuitpython-ble

# Clone the repository (Assumes Git for Windows is installed)
git clone https://github.com/5teveLAN/indoor-positioning-bilateration.git

# Navigate to the specific directory
cd .\indoor-positioning-bilateration\beacons
```
